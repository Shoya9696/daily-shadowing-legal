プライバシーポリシー

Daily Shadowing（デイリー・シャドーイング）

1. アプリ名

Daily Shadowing（デイリー・シャドーイング）

2. 適用範囲

本プライバシーポリシー（以下「本ポリシー」）は、運営者が提供するモバイルアプリケーション「Daily Shadowing」（以下「本アプリ」）の利用に関して適用されます。

本ポリシーは、Free 機能および Premium（有料サブスクリプション）機能を含む現行版を対象とします。サービス内容の変更がある場合は、本ポリシーを改定します。

3. 独自アカウントについて

本アプリは、独自のユーザー登録・ログイン機能を提供しません。氏名・パスワード等を入力するアカウント作成画面はありません。

iOS の Premium 購入・復元では、Apple のアカウントおよび課金基盤（StoreKit）と RevenueCat を利用します。RevenueCat は、独自ログインなしでも動作する匿名の App User IDを用いて、購入状態（エンタイトルメント）を管理します。

4. 運営者が取り扱う・取り扱わない情報

4.1 問い合わせによりお送りいただく情報

サポート用メール（daily.shadowing.app@gmail.com）へご連絡いただく場合、メールアドレス、お名前（任意）、端末情報、OS、アプリバージョン、問い合わせ内容など、あなたが記載した情報が運営者に届きます。これらは問い合わせ対応の目的でのみ使用します。

4.2 課金・サブスクリプション関連（iOS）

Premium の購入・復元・購読状態の確認にあたり、次の処理が行われます。

  • Apple StoreKit：Apple の課金システムによる購入処理。クレジットカード番号などの支払い手段は Apple が処理し、運営者がクレジットカード番号を取得・保管することはありません。
  • RevenueCat：購読状態の管理・購入復元のために利用します。RevenueCat は匿名 App User ID、購入・エンタイトルメント情報、端末・OS・アプリに関する技術情報などを処理する場合があります。詳細は RevenueCat のプライバシーポリシーもご参照ください。

運営者は、課金可否や Premium 状態をアプリ上で反映するために、RevenueCat / Apple 経由で得られる購読関連情報を利用します。運営者が決済カード情報を直接取得する仕組みはありません。

5. 利用目的

  • 英語シャドーイング学習機能の提供（教材表示、音声再生、語彙表示等）
  • Free / Premium の機能差に基づくアクセス制御（今日の教材、過去教材、バックグラウンド再生等）
  • 購入・復元・サブスクリプション状態の確認
  • 問い合わせ対応
  • サービス改善・不正防止・障害対応（必要な範囲）

6. 外部送信・第三者サービス

6.1 教材・音声の配信

教材 JSON および音声ファイルは、GitHub Pages 上のコンテンツ配信から取得します。アプリは学習のためにこれらのデータを端末へダウンロードします。あわせて、端末内に offline JSON cache（オフライン用のキャッシュ）を保持する場合があります。キャッシュは利便性のためのものであり、すべての教材・音声の完全なオフライン利用を保証するものではありません。

6.2 RevenueCat / Apple

上記 4.2 のとおり、課金関連の処理のために Apple および RevenueCat のサービスと通信する場合があります。

6.3 Firebase

本アプリには Firebase Core / Firebase Auth の依存関係が含まれています。ただし、独自のログイン UI は提供しておらず、ユーザーが操作して認証情報を送信する導線はありません。Firestore、Firebase Storage、Cloud Functions、firebase_analytics は使用していません。

6.4 単語読み上げ（TTS）

語彙の読み上げには端末 OS の TTS 機能を利用することがあります。読み上げ処理は端末上で行われ、運営者がユーザーの声を収集するものではありません。

7. 第三者提供

運営者は、法令に基づく場合を除き、問い合わせメール等であなたから受領した個人情報を、本人の同意なく第三者に販売・提供しません。

課金処理のために Apple および RevenueCat が各自のサービスとして情報を処理する場合があります。各社の取扱いは各社のポリシーに従います。

8. 音声・録音データ

  • 録音機能はありません
  • ユーザーの声を録音・収集・送信する機能はありません
  • マイク権限は要求しません

9. 教材データ

  • 英文は運営者が作成したオリジナルコンテンツです
  • 外部ニュース記事本文・外部教材の転載は行っていません
  • 音声は運営者が TTS 等で生成し、GitHub Pages 経由で配信します

10. 子どもの利用

本アプリは一般向けの英語学習アプリです。13 歳未満のユーザーを主要な対象として設計していません。子どもが利用する場合は、保護者の監督のもとでご利用ください。

11. ユーザーの権利

問い合わせメール等で運営者が保有する個人情報について、開示・訂正・削除等のご請求がある場合は、下記問い合わせ先までご連絡ください。合理的な範囲で対応します。

Apple アカウント上の購入履歴・サブスクリプション管理については、Apple の手続に従ってください。

12. 保存期間・削除

12.1 運営者が受領する問い合わせ情報

サポート用メール（daily.shadowing.app@gmail.com）へお送りいただいた内容は、問い合わせ対応および必要に応じた記録のため、対応完了後も合理的な期間保存する場合があります。

保存している問い合わせ情報の削除をご希望の場合は、同メールアドレス宛に削除依頼を送信してください。本人確認に必要な範囲でご協力をお願いする場合があります。対応可能な範囲で、遅滞なく削除その他適切な措置を行います。

12.2 端末内の offline JSON cache

本アプリは、教材 JSON 等を端末内にキャッシュする場合があります。キャッシュの削除・消去は、次の方法で行えます。

  • 端末の設定から本アプリを削除（アンインストール）する
  • iOS の場合、端末のストレージ設定等から本アプリのデータを削除する（OS の表示・手順に従います）

アプリの再インストール後は、再度ネットワーク経由で教材・音声を取得する必要があります。

12.3 Apple / RevenueCat

Apple および RevenueCat が各自のサービスとして処理・保存する情報の保存期間・削除手続は、各社のプライバシーポリシーおよび手続に従います。運営者がそれらの保存期間を直接制御するものではありません。

13. 第三者サービスの選定

運営者は、Apple、RevenueCat、GitHub Pages 等の第三者サービスを利用するにあたり、提供される情報保護の水準が本ポリシーの趣旨に照らし同等以上であることを確認したうえで利用します。各社の取扱いの詳細は、各社のポリシーをご確認ください。

14. ポリシー変更

運営者は、法令やサービス内容の変更に応じて本ポリシーを変更することがあります。重要な変更がある場合は、アプリ内通知やストア掲載情報等でお知らせします。
