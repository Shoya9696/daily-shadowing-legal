プライバシーポリシー

Daily Shadowing（デイリー・シャドーイング）— 無料β版

1. アプリ名

Daily Shadowing（デイリー・シャドーイング）

2. 適用範囲

本プライバシーポリシー（以下「本ポリシー」）は、運営者が提供するモバイルアプリケーション「Daily Shadowing」（以下「本アプリ」）の利用に関して適用されます。

本ポリシーは、無料β版を対象とします。将来、有料機能・ログイン・広告等を追加する場合は、本ポリシーを改定します。

3. 取得する情報

現時点の無料β版において、運営者はユーザーから氏名・メールアドレス・電話番号などの個人情報を取得しません。

本アプリにユーザー登録画面はなく、ユーザーが個人情報を入力するフォームも提供していません。

4. 取得しない情報

無料β版では、以下の情報を意図的に取得しません。

区分内容
登録情報氏名、メールアドレス、電話番号、住所等
音声・録音ユーザーの声の録音データ（録音機能なし）
学習採点AI 採点結果、発話解析データ（AI 採点機能なし）
位置情報GPS 等による位置情報
連絡先連絡先一覧
メディア写真、カメラ、マイクによる収集データ
課金情報クレジットカード番号等（実課金機能なし）
広告識別子広告 SDK によるトラッキングデータ（広告なし）
行動分析解析 SDK による利用行動の送信（解析 SDK なし）

5. 利用目的

無料β版では、ユーザーから個人情報を取得しないため、個人情報の利用目的は該当しません。

本アプリは、端末内に同梱された教材データを用いて、英語シャドーイング学習を支援することを目的とします。

6. 第三者提供

運営者は、無料β版において、ユーザーの個人情報を第三者に提供しません。

7. 外部送信

教材・音声データ

教材テキスト・音声ファイルはアプリ内に同梱されており、学習のために端末上で読み込まれます。ユーザーが学習する内容を運営者のサーバーへ送信する機能はありません。

Firebase について

本アプリには、将来機能のため Firebase Core / Firebase Auth の依存関係および設定が含まれています。

ただし、無料β版では次のとおりです。

  • ユーザー登録・ログイン画面を提供していません
  • ユーザーが操作して Firebase Auth に認証情報を送信する導線はありません
  • Firestore、Firebase Storage、Cloud Functions は使用していません
  • firebase_analytics は未導入（依存関係に含まれていません）
  • 広告 SDK・行動分析 SDK は未導入です

アプリ起動時に Firebase Core の初期化処理が実行される場合がありますが、無料β版ではログイン UI を提供しておらず、ユーザーが入力した個人情報を運営者が取得する機能はありません。

単語読み上げ（TTS）

語彙の読み上げには、端末 OS の TTS（Text-to-Speech）機能を利用します。読み上げ処理は端末上で行われ、運営者がユーザーの音声データを収集するものではありません。

8. 広告・解析ツール

無料β版では、以下を使用していません。

  • 広告 SDK（AdMob 等）
  • 課金 SDK（RevenueCat 等）
  • 行動分析 SDK（firebase_analytics、Google Analytics 等）※いずれも未導入
  • クラッシュレポート SDK（Crashlytics 等）※導入時は本ポリシーを改定

9. ログイン・認証

無料β版では、ログイン UI を表示せず、ユーザー登録を行いません。

Firebase Auth 関連のコードは将来の Premium・同期等のためにリポジトリに含まれる場合がありますが、無料β版の通常利用では認証機能は提供されません。

10. 音声・録音データ

  • 録音機能はありません
  • ユーザーの声を録音・収集・送信する機能はありません
  • マイク権限は要求しません

11. 教材データ

  • 英文は運営者が作成したオリジナルコンテンツです
  • 外部ニュース記事・外部教材の転載は行っていません
  • 音声は運営者が TTS で生成したものを同梱しており、外部音声素材の転載は行っていません

12. 子どもの利用

本アプリは、英語学習を目的とした一般向けアプリです。13 歳未満のユーザーを主要な対象として設計していません。

子どもが利用する場合は、保護者の監督のもとでご利用ください。

将来、グローバル展開や子ども向け（Kids）提供を行う場合は、各国の法令（COPPA 等）に応じて本ポリシーを見直します。

13. ユーザーの権利

無料β版では、運営者がユーザーの個人情報を保有しないため、開示・訂正・削除等の請求の対象となる個人情報は原則として存在しません。

14. ポリシー変更

運営者は、法令の変更やサービス内容の変更に応じて、本ポリシーを変更することがあります。重要な変更がある場合は、アプリ内通知やストア掲載情報等でお知らせします。
